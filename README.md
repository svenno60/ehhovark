# ehhovark

## Deployment on GitHub Pages

To deploy the application on GitHub Pages, follow these steps:

1. Create a new repository on GitHub and push your code to the repository.
2. In the repository settings, go to the "Pages" section.
3. Select the branch you want to deploy from (e.g., `main`) and the folder (e.g., `/root`).
4. Save the settings, and your application will be deployed on GitHub Pages.

## Running the Flask Application Locally

To run the Flask application locally, follow these steps:

1. Install the required dependencies by running:
   ```
   pip install -r requirements.txt
   ```
2. Run the Flask application:
   ```
   python app.py
   ```
3. Open your web browser and go to `http://127.0.0.1:5000` to access the application.
